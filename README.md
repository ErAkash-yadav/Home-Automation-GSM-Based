📌 GSM Based Home Automation System
GSM Based Home Automation System is an electronics project that allows users to control home appliances remotely using a mobile phone through GSM technology. By sending an SMS or making a call, users can turn electrical devices ON or OFF from anywhere.
This project uses a GSM module and microcontroller to receive commands from the user and control appliances through relay switches. It is a cost-effective and reliable solution for remote home automation, especially in areas where internet connectivity is limited.
🚀 Features
Control home appliances using SMS or phone call
Works without internet (uses GSM network)
Simple and low-cost automation solution
Remote access from anywhere
Easy to install and expand
⚙️ Components Used
Microcontroller (Arduino / ATmega / similar)
GSM Module (SIM800 / SIM900)
Relay Module
Power Supply
Connecting wires
Home appliances (bulb, fan, etc.)
🔧 How It Works
The user sends an SMS command from a mobile phone.
The GSM module receives the message.
The microcontroller processes the command.
According to the command, the relay turns the appliance ON or OFF.
📱 Example Commands
LIGHT ON
LIGHT OFF
FAN ON
FAN OFF
